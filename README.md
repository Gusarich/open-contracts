# math.func
`math.func` is a math library that extends FunC's arithmetic operations.

## Usage
```c

() main(){
    int n1 = exp(2); ;; or 2.exp()
    n1~dump(); ;; 4

    int n2 = sqrt(36); ;; or 36.exp()
    n2~dump(); ;; 6
}
```

## Functions
- `(int) sqrt(int x)`
- `(int) avg(int x, int y)`
- `(int) exp(int x)`
- `(int) log2(int x)` 
- `(int) mod (int x, int y)`
